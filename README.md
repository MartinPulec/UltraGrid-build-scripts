# UltraGrid-build-scripts

As the builds are now hosted by GitHub CI, there are now just supplementary
build scripts for (subdir names):

- centos7_docker - Linux AppImage build for old Linuxes (https://ces.net/UltraGridAltBuilds - UltraGrid-glibc2.17-x86_64.AppImage)
- macOS-10.15 - https://github.com/CESNET/UltraGrid/releases/download/continuous/UltraGrid-nightly-alt.dmg
- macOS - https://ces.net/UltraGridAltBuilds - UltraGrid-legacy.dmg (may not be rebuilt any more)

This repository contain helper scripts to build and deploy
[UltraGrid](https://github.com/CESNET/UltraGrid) for these platforms
but those are **NO LONGER USED** :
- Windows/MSYS
- macOS
- Linux (AppImage)
